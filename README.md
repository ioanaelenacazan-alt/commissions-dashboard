# App Comisioane

Aplicație web construită în React pentru calculul rapid al comisioanelor pentru consultanți și team leaderi, pe baza datelor importate din fișiere Excel.

---

## 🚀 Funcționalități

- Upload fișier Excel
- Calcul automat comisioane (TM6, accesorii, demo)
- Adăugare manuală persoane
- Editare și ștergere persoane
- Persistență date (LocalStorage)
- Filtrare după:
  - lună
  - consultant
  - team leader
- Export PDF
- Dark Mode / Light Mode

---

## 🧠 Tehnologii folosite

- React (useState, useMemo, custom hooks)
- Tailwind CSS
- LocalStorage
- jsPDF
- Structură modulară (components / hooks / utils)

---

## 📂 Structură proiect

- `components/` – componente UI
- `hooks/` – logică reutilizabilă
- `utils/` – funcții helper
- `constants/` – date statice
